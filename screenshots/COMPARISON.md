# Discord Style Design Comparison

This document compares the UI before and after applying Discord-style design guidelines.

## Design Changes Summary

| Element | Before | After |
|---------|--------|-------|
| Primary Color | Indigo (#4F46E5) | Discord Blurple (#5865F2) |
| Button Style | Rounded (12px) | Pill-shaped (20px) |
| Input Fields | Rounded (12px) | Subtle corners (4px) |
| Chips | Rounded (8px) | Pill-shaped (full) |
| FAB | Rounded square (16px) | Circular |
| Headline Font | Poppins | Outfit |
| Body Font | Inter | Inter |
| Background | Light gray (#FAFAFC) | Pure white (#FFFFFF) |
| Card Background | Light blue-gray | Light gray (#EBEDEF) |

## Key Design Principles Applied

1. **Discord Blurple (#5865F2)** - Primary brand color throughout
2. **Pill-shaped buttons** - Modern, recognizable Discord style
3. **Clean white backgrounds** - Spacious, minimal feel
4. **Subtle input corners** - Less aggressive rounding (4px)
5. **Circular FAB** - Discord-style floating action button
6. **High contrast text** - Dark gray (#2E3338) on white
7. **Outfit + Inter fonts** - Clean, modern typography
